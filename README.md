### 浏览器插件安装
1. 下载包解压
2. 在 Chrome 地址栏输入 `chrome://extensions/` 以打开扩展程序管理页面
3. 顶部勾选“开发者模式”，再点击“加载已解压的扩展程序...”，路径为刚才的解压目录

### 客户端安装
- mac:下载对应操作系统的安装包，正常安装即可。
- windows免安装:下载解压打开downloader.exe
